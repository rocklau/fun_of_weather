# 天气与气候模拟

使用 **Three.js** 构建的交互式 3D 天气和气候模拟集合。这些项目在基于浏览器的 3D 环境中可视化复杂的大气和海洋现象。

## 项目

### 1. 云层模拟器 (`cloud.html`)
**"云的谱系：风场与演化可视化"**

一个专注于云的生命周期和大气动力学的交互式模拟。

**功能特性：**

*   **云的演化**: 可视化云在不同阶段的转变：
    *   💧 **水蒸气**: 从地面升起的不可见水蒸气
    *   ☁️ **淡积云**: 晴天云
    *   🥦 **浓积云**: 高耸的积云，表明不稳定
    *   ⛈️ **积雨云**: 成熟的风暴云，带有砧状云顶
    *   🌊 **层积云**: 分层的低云
    *   🌫️ **层云**: 无特征的低云层
    *   🪶 **密卷云**: 高空冰晶云
*   **风场系统**: 使用粒子轨迹动态可视化气流、湍流和风切变
*   **交互式飞机**: 对大气条件（湍流、上升气流）做出反应的 3D 飞机模型
*   **实时监控**: 显示垂直对流、水平风切变和高度的仪表板

### 2. 海洋气候系统 (`ocean_climate.html`)
**"海洋气候系统 - 大陆边缘模拟"**

大陆边缘气候相互作用的综合模拟，具有昼夜循环和海陆相互作用。

**功能特性：**

*   **地形与海洋**: 逼真的 3D 地形，代表深海、大陆坡、大陆架和海岸山脉
*   **动态气候引擎**:
    *   **昼夜循环**: 影响温度的真实太阳运动
    *   **海陆风**: 基于陆地和海洋之间温差模拟风向反转
    *   **水循环**: 可视化蒸发、云形成和降水
    *   **上升流**: 模拟富含营养的深层水上升到表面
*   **场景模式**:
    *   **正常**: 平衡的气候与信风
    *   **风暴**: 高能量状态，伴有强降雨和强风
    *   **干旱**: 高压、低蒸发和干燥条件
    *   **上升流**: 强离岸风驱动深海上升流
*   **数据监控**: 实时显示蒸发率、风速/风向、降水量和上升流强度的仪表

## 使用技术

*   **[Three.js](https://threejs.org/)**: 3D 渲染引擎
*   **[Tween.js](https://github.com/tweenjs/tween.js)**: 平滑动画过渡
*   **HTML5/CSS3**: UI 覆盖层和布局

## 如何运行

这些模拟完全在浏览器中运行，无需构建步骤。

1.  克隆或下载此仓库
2.  在现代浏览器（Chrome、Firefox、Safari、Edge）中直接打开 `cloud.html` 或 `ocean_climate.html`
3.  **注意**: 为获得最佳体验，请使用本地 Web 服务器（例如 VS Code Live Server、`python -m http.server`）以避免纹理加载时可能出现的 CORS 问题，尽管这些特定文件主要使用程序化纹理。
